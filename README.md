# labProject_v3

A static frontend project demonstrating a shopping basket interface with HTML, CSS, and vanilla JavaScript.

## 📋 Overview

This project showcases a responsive e-commerce shopping experience with:
- Product catalog page with browseable items
- Functional shopping basket with add/remove operations
- Clean, modern UI with custom styling
- No external dependencies or build process required

## 📁 Project Structure

```
labProject_v3/
├── index.html              # Product catalog page
├── basket.html             # Shopping basket page
├── assets/
│   ├── css/
│   │   ├── style.css       # Main page styles
│   │   └── basket.css      # Basket page styles
│   ├── images/             # Product images

│   │   ├── c2.avif

```

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Edge, Firefox, Safari)

### Running the Project

1. **Open the main page:**
   - Double-click `index.html`, or
   - Right-click → "Open with" → Select your browser, or
   - Drag the file into your browser window

2. **Navigate to the basket:**
   - Use the UI buttons/links on the main page, or
   - Open `basket.html` directly in your browser

## 💻 Development

### Tech Stack
- **HTML5** — Semantic markup
- **CSS3** — Modern styling (including AVIF images)
- **Vanilla JavaScript** — No frameworks or dependencies

### Notes
- This is a static site — no build step, server, or installation required
- Edit HTML files directly in any text editor
- CSS changes take effect on page refresh
- JavaScript modifications update without any tooling

### File Editing Tips
- Styles: Modify `assets/css/style.css` (main page) or `assets/css/basket.css` (basket page)
- Functionality: Edit `assets/js/script.js` (main page) or `assets/js/b-script.js` (basket page)
- Markup: Update `index.html` or `basket.html` directly
